# NeuralHive ML Mondays Blog (Starter)

This is a starter scaffold for the **Neural Hive WebDev Recruitments Challenge – ML Mondays Blog**.

It focuses on **file structure** and minimal boilerplate using **Astro** as the Static Site Generator (SSG).

## Tech Stack

- [Astro](https://astro.build/) – Static Site Generator
- Plain CSS (you can later add Tailwind, etc.)

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the dev server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

4. Preview the build:

   ```bash
   npm run preview
   ```

## Pages Included

- `/` – Home
- `/about` – About ML Mondays
- `/posts` – All posts listing
- `/ethos` – Ethos / principles
- `/write-for-us` – Write for Us
- `/posts/[slug]` – Individual blog post page

Content lives in `src/content/posts` as Markdown files.

You can now focus on:
- Improving the design
- Implementing writer workflow
- Adding SEO, analytics, and deployment
